## 🏗️ What I Do

I design **audit-first systems** for high-friction markets — government transparency, electoral validation, agro-industrial traceability, and autonomous robotics. My work lives at the intersection of **civic technology**, **AI agent infrastructure**, and **open-source tooling**.

I created the **Gentleman AI ecosystem** — tools that teach coding agents to mentor, remember, and review. Not chatbots. Not wrappers. Real infrastructure. 15 supported agent platforms, 10k+ combined GitHub stars, and a philosophy that AI should **elevate** engineers, not replace them.
